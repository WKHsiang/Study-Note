# 脚手架工具

## 1. Yeoman
   
### 1-1. 安装
- 在全局范围安装yo  
```
$ npm install yo --global
```

- 安装对应的generator

``` 
$ npm install generator-node --global
```
 
- 通过yo运行generator
```
$ cd path/to/project-dir
$ mkdir my-module
$ yo node
```   

### 1-2. 使用步骤 
1. 明确自己的需求
2. 找到合适的Generator
3. 全局范围安装找到的Generator
4. 通过Yo运行对应的Generator
5. 通过命令行交互填写选项
6. 生成自己所需要的项目结构
